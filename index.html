Capabilities

Terminal Superpowers

Everything you need to search, browse, and stream anime — right from your command line.

01

Interactive TUI

Two-column search layout with instant metadata preview. Navigate with vim keys, built on Bubble Tea v2's Elm architecture.

02

Multi-provider Search

Parallel search across Jikan and AniList APIs. Results merged with unified ranking for comprehensive coverage.

03

Quality Selection

Cycle through presets: best, 1080p, 720p, 480p, 360p, auto. Adaptive streaming with instant quality switching.

04

Sub / Dub Toggle

Switch between subtitled and dubbed versions on the fly with Ctrl+T. Seamless transition between audio tracks.

05

Auto-Skip Intro / Outro

AniSkip integration with mpv Lua script. Automatically jumps intros and outros using community-sourced timestamps.

06

Multi-host Extraction

Parallel stream extraction from 8+ hosts with priority fallback. First successful result wins with 500ms grace period.

07

Android Proxy

Local HTTP proxy for seamless playback on Termux/PRoot. Custom DNS resolution bypasses platform limitations.

08

Persistent Caching

SQLite-backed ID mapping for instant repeated lookups. MAL ID to AllAnime ID resolution cached permanently.

Get Started

Install Anilix

One command. Works on Linux, macOS, Windows, and Android (Termux).

Linux / macOS / Termux

curl -fsSL https://raw.githubusercontent.com/hishantik/anilix/main/install.sh | sh

Windows (PowerShell)

Invoke-WebRequest -Uri "https://raw.githubusercontent.com/hishantik/anilix/main/install.bat" -OutFile "install.bat"; .\install.bat

Go Install

go install github.com/hishantik/anilix@latest

Build from Source

git clone https://github.com/hishantik/anilix && cd anilix && go build -o anilix .
